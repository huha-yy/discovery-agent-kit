## Repository About

### Short description

`Reusable agent skills, workflows, and templates for Discovery-style multi-cloud resource discovery projects`

### Longer homepage blurb

Discovery Agent Kit packages reusable agent methods extracted from enterprise multi-cloud discovery work. It helps teams standardize account onboarding, pack development, SOW auditing, real-cloud validation, runtime troubleshooting, and document closure across projects.

## First Public Release

### Release title

`discovery-agent-kit v0.2.0`

### Release body

This release prepares `discovery-agent-kit` for broader reuse across teams and repositories.

Included in `v0.2.0`:

- reusable skills for account onboarding, pack development, SOW auditing, document sync, and release review
- new reusable skill: `discovery-real-cloud-validation`
- new reusable skill: `discovery-runtime-troubleshooting`
- matching workflows for validation and troubleshooting
- matching templates for validation planning and runtime incident summaries
- top-level publishing assets:
  - `LICENSE`
  - `PUBLISHING.md`
  - updated `README.md`
  - updated `RELEASE-NOTES.md`

Highlights:

- separate reusable agent knowledge from project-private runtime evidence
- give teams a repeatable method for real-cloud acceptance and operational triage
- make the repository ready for GitHub publication with clear versioning and release guidance

Recommended use:

- install skills into a local Codex-compatible environment
- adapt workflows and templates per project
- keep customer-specific keys, screenshots, and runtime evidence out of this shared repository
